# Weather Prediction using Machine Learning

## 📌 Overview
This project uses machine learning to predict weather conditions for the next 24 hours based on historical data.

## 🚀 Features
- Predicts temperature, humidity, wind speed, and precipitation  
- Uses Linear Regression for forecasting  
- Flask-based API for serving predictions  
- 24-hour hourly predictions  

## 🛠️ Tech Stack
- Python  
- scikit-learn  
- Flask  
- Pandas, NumPy  

## 📊 Dataset
- Kaggle Weather Dataset  

## ⚙️ Methodology
- Data preprocessing (handling missing values, normalization)  
- Feature engineering (time-based features)  
- Train-test split (80/20)  
- Separate models for each weather parameter  

## 📈 Evaluation
- Mean Absolute Error (MAE)  
- Root Mean Squared Error (RMSE)  

## ▶️ How to Run
1. Install dependencies  
2. Run Flask server  
3. Input location to get predictions  

## 📌 Future Improvements
- LSTM / Deep Learning models
